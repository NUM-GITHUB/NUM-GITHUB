GCN example code:

In [2], the GCN code is shown as below

Even though this is not a very typical implementation of GCN, it shows all the stages of GCN.

Firstly, at initialization stage, it initialized a linear neural network layer. In this case, dim_in and dim_out are same value and equals to the number of features of for given node. In forward function, the node features (n×d) is firstly passed to the linear layer and then passed through a Relu activation function, which can be written as σ(WXl ) (n×d). Then, it is multiplied with adjacency matrix (n×n) Aσ(WXl )=σ(AWXl ) (n×d). Finally, it pass through another activation function to get the final value, σ(AWXl ) (n×d).
